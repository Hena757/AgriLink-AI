# AgriLink-AI
AI-powered smart farming assistant prototype

## Problem Statement
Farmers face crop losses due to late disease detection, weather unpredictability, and lack of timely guidance.

## Our Solution
AgriLink AI provides:
- AI chat support
- Crop disease identification
- Weather-based farming recommendations
- Offline accessibility
