## Problem Statement
Farmers suffer losses due to late disease detection, unpredictable weather,
and lack of real-time expert guidance.

## Solution
AgriLink AI is a smart digital companion that provides:
- AI-powered farming guidance
- Crop disease detection (image-based)
- Weather-aware recommendations
- Offline-ready & scalable design

## Prototype Features
- Interactive AI chat (prototype)
- Crop disease identification (demo)
- Weather alerts based on location
- Farmer-friendly UI

## Tech Stack
- Frontend: Streamlit
- Backend Logic: Python
- AI (Prototype): Rule-based / simulated AI
- Hosting: Streamlit Community Cloud

## System Workflow
1. Farmer selects service
2. Inputs query / image
3. AI analyzes input
4. Recommendations shown instantly
